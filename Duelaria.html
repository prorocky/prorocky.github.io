How to set up a room

Here's how to set up a Duelaria lobby

Keep in mind this is a 2-player game so you will need 2 people

First screen

After clicking Run Game, you will be greeted with this screen

Lobby Creation

After a brief wait, you will arrive to this screen with Create and Join buttons

Creation

One player types a lobby name into the top textbox and then clicks Create

Joining

The other player types the same lobby name in the lower textbox and click Join

Waiting

Both players have this screen until they both load into the game

Duel!

Battle it out to the death!
If any issues arise on any step, please refresh the page and try again

Controls

Simply use the A/D keys for movement, spacebar for jumping, mouse for aiming, and left click for shooting
